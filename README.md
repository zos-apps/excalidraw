## About

Virtual whiteboard for sketching and diagramming.

## Features

- Freehand drawing
- Shapes
- Text
- Collaboration
- Export options

## Installation

### From App Store
Open the **App Store** in zOS and search for "Excalidraw".

### Via npm
```bash
npm install @zos-apps/excalidraw
```

## Usage

- Select tool from toolbar
- Draw on canvas
- Export to PNG/SVG

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `V` | Select |
| `R` | Rectangle |
| `D` | Diamond |
| `E` | Ellipse |
| `A` | Arrow |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
